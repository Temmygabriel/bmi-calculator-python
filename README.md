# BMI Calculator (Python)

This project is a simple Body Mass Index (BMI) calculator built using Python
and Jupyter Notebook.

## Features
- Accepts weight in kilograms
- Accepts height in feet and inches
- Converts height to meters
- Calculates BMI
- Classifies BMI into categories

## Technologies Used
- Python
- Jupyter Notebook

## How to Run
1. Open the notebook in Jupyter Notebook
2. Run all cells
3. Enter your weight and height when prompted
